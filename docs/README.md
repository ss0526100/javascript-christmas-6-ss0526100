  - [ ] 예외 체크 (`[ERROR] {예외 내용}. 다시 입력해주세요`)

      - [ ] 메뉴의 개수가 `1` 이상의 숫자인가(model)

      - [ ] 중복 메뉴인가(model)
    - [ ] (`메뉴는 총 20개까지만 주문할 수 있습니다.`)
      - [ ] 모든 메뉴 개수가 20개 이하인가(model)

      - [ ] 음료만 주문하지 않았는가 (model)
  - [ ] 예외 값일 경우 다시 입력받기
